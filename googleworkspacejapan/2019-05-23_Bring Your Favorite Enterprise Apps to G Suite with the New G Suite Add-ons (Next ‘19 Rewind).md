## 文字起こし

2019年には約500のセッションが予定されているため、
すべてに参加できなかったのは当然です。G
Suiteアドオンについてまだご存知ない方は、ぜひご注目ください。
ここでは、
500万以上の企業が生産性の中核としてG Suiteを使用しており、
多くの場合、Gmail、Drive、ドキュメント、カレンダーを1日に何度も使用しています。しかし、
実際には仕事はそれぞれ異なり、
G Suiteは
仕事をするために必要な唯一のアプリケーションではありません。仕事の固有のニーズに
対応するには、多くの専用アプリが必要です。
しかし、そのためには、
G Suiteと他のアプリの間を行ったり来たりして
仕事をこなす必要があります。このアプリの切り替えは積み重なり、
コンテキストの
切り替えが人々の
生産性を著しく低下させるという研究結果が出ています。幸いなことに、
G Suiteはコンテキストの切り替えを
減らすことに注力しており、特に
新しいクイックアクセスサイドパネルでは、
1つのウィンドウで2つのG Suiteアプリを操作できます。これ
までは、
サイドパネルを使用して他のGoogleアプリにアクセスするしかできませんでしたが、
最近、
お気に入りのアプリをG Suiteに取り込むことができるG Suiteアドオンを発表しました。それだけで
なく、コンポーズとも統合できます。
Gmailのボックス、ドライブの右クリックメニュー、カレンダーの
会議オプションなど、
もちろん
今後さらに多くのアクセスポイントが計画されています。G
Suiteアドオンを使用すると、
主要なソフトウェアベンダーのアプリに、すべての
主要なG Suiteアプリのサイドパネルから直接アクセスできます。
作業中にこのアドオンを開いたままにしておくことで、コンテキストの切り替えを減らし、
生産性を向上させることができます。G Suite
アドオンは、アドオン内で関連情報を表示し、
作業の完了に役立つアクションを実行できるようにします。
たとえば、メールスレッドで顧客の詳細情報を確認できます。
アドオンは、セキュリティが
確保され、適切に管理されている場合にのみ企業にとって有用であることを私たちは十分に認識しています。その
ため、G Suiteアドオンはデフォルトで安全であり、
アクセスが制限されています。つまり、
限られた期間のみ限られたデータを許可するスコープです。たとえば、G Suite
アドオンは、カレンダー全体への無制限のアクセスではなく、一度に1つの会議のカレンダーデータへのアクセスのみを許可し、
ユーザーがその会議を表示している間のみ
アクセスを許可します。
同様に、G Suiteアドオンは、
Google管理コンソールで管理者によって完全に管理できます。当社のツールを使用すると、
管理者は選択したアドオンを自動的にインストールしたり、
ユーザーのみがアドオンをインストールできるようにしたりできます。
管理者チームによって承認された
権限 API ホワイトリスト
ツールも提供しています。これにより、管理者は、アドオンを
含むサードパーティ アプリケーションがアクセスできる API とアクセスできない API を定義できます。G
Suite アドオンには、開発者ツール、
セキュリティ監査、活気のあるマーケットプレイスなど、他にもたくさんの素晴らしい機能がありますが、
