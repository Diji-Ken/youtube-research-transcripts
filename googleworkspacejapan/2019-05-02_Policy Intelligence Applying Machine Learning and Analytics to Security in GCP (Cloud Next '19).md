## 概要
Cloud environments today are complex. Even when all the information is available, it’s hard to pull it all together to make a good decision. Imagine if your cloud could make your cloud environment more efficient and secure with a push of a button.

Come learn about how Google Cloud is using machine learning and data analytics to help save customers money and make them safer.
