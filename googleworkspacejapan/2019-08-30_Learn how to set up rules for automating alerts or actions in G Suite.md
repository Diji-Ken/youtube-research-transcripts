## 概要
You can establish rules for automating alerts or actions in G Suite with the Activity Rules section. This installment of the @Google Cloud Security Showcase explains how you can create custom rules that monitor different events and data sources. For example, you may want to create a late spam classification rule in which @Google detects and reclassifies spam after it’s delivered.
