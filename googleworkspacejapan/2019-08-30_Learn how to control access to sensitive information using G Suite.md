## 概要
With G Suite, you can control who is able to access sensitive information based on the device or user. In this installment of the @Google Cloud Security Showcase, you’ll learn how to use G Suite’s Content Aware Access tool, which lets you create access levels based on different attributes such as OS version, IP subnets, and more.
