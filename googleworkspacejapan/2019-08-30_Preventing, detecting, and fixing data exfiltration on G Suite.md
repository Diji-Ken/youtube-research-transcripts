## 概要
Learn about data exfiltration in this installment of the @Google Cloud Security Showcase. You’ll start off learning how to prevent data exfiltration using G Suite’s data loss prevention product. Then, you’ll discover how to detect and remediate data exfiltration using the investigation tool within the G Suite Security Center.
