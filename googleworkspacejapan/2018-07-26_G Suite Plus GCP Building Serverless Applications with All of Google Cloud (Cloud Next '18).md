## 概要
A lot of enterprises rely on G Suite tools like Gmail, Drive, Docs, Sheets, and Calendar to collaborate in real time and get things done. If you're using G Suite at work, your organization may have a ton of untapped G Suite data. Unlock data insights using the GCP’s data warehousing tools like BigQuery or extend your GCP applications with G Suite’s powerful collaborative and reporting capabilities using Google Cloud serverless technologies. Join us for this informative session where you will learn how you can build serverless applications on top of G Suite and GCP.
