## 概要
Python×Messaging API×GitHub ActionsでLINEチャットボットを作成します！
今回は、毎朝美女からメッセージが届くチャットボットの作成になります。
LINEが提供するMessaging APIを用いてチャットボットの土台を作り、最新のワークフロー自動化ツールでもあるGitHub Actionsを用いて定期実行する仕組みを作りました！
非常に実践的かつ面白い内容となっております！
